기술적 특징 및 도전 과제
================================

1. URL 구조 분석
- 목록 페이지: `/www/selectBbsNttList.do?bbsNo=3&key=23`
- 상세 페이지: `/www/selectBbsNttView.do?key=23&bbsNo=3&nttNo={id}&...`
- 첨부파일: `/www/downloadBbsFile.do?atchmnflNo={fileId}`

2. 주요 기술적 도전
- 상대 경로 URL 처리: 링크가 `./selectBbsNttView.do`로 시작하여 절대 경로 변환 필요
- 한글 파일명 지원: Content-Disposition 헤더에서 한글 파일명 추출
- HTML 구조: 테이블 기반 레이아웃으로 컨텐츠 추출

3. 해결 방법
- URL 조인: urljoin()을 사용하여 `./` 접두사 제거 후 기본 URL과 결합
- 파일명 인코딩: unquote()로 URL 디코딩 처리
- DOM 파싱: BeautifulSoup으로 테이블 구조 분석
